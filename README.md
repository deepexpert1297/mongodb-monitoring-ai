# MongoDB Monitoring with AI

## 📌 Overview
Monitor MongoDB logs and generate alerts with AI-based analysis.

## 🔥 Features
- Real-time log monitoring
- AI-based issue detection
- Automated alerting

## 🛠️ Tech Stack
- MongoDB
- Node.js
- Axios
- OpenAI API

## ⚙️ Setup & Installation

### 🔹 Prerequisites
- Linux system (Ubuntu / Amazon Linux)
- Node.js (v18+)
- MongoDB installed
- npm package manager

---

### 🔹 Install Dependencies

```bash
npm install
```

## 🚀 Usage
- Monitor mongod.log
- Send logs to AI
- Get solutions automatically
